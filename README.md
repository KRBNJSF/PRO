# PRO

a++ nebo a-- je postfixová operace, což znamená, že hodnota a se po vyhodnocení výrazu změní.<br>
++a nebo --a je operace s prefixem, což znamená, že hodnota a se před vyhodnocením výrazu změní.<br>
<b>Object</b> - V programu reprezentuje reálnou věc. Objekt data seskupuje v paměti. Každý objekt má své vlastní atributy a metody podle vzoru (třídy). Atributy definující objekt však nemusí mít nutně pevné hodnoty. Metody s daty pracují.<br>
<b>Třída</b> - Vzor podle kterého se vytváří objekt v paměti. Dá se říct, že je to základní prvek OOP sloužící jako předpis pro vznik objektu jako instance třídy. Třída definuje vlastnosti (atributy) a metody objektů, které při běhu programu vzniknou. Hodnoty vlastností se mohou u jednotlivých objektů odlišovat, na každém objektu, který vznik z jedné třídy, můžeme zavolat pomoci shodnou sadou metod.<br>
<b>Konstruktor</b> - Druh metody, která se zavolá k okamžiku vytváření nové instance. Jedná se o metodu, která sestaví objekt/inicializuje objekt. při inicializaci, tj. vzniku objektu, musí být zadány hodnoty atributů jméno a věk<br>
<b>Getter/Setter</b> - Pomocné metody pro získání nebo nastavování hodnot v objektu. Složí pro kontrolu/validaci nastavovaných dat. Jsou to metody, které bude možné na instanci - objektu - použít (zavolat)<br>
<b>Static/non-static</b> - Statické metody jsou vázané na třídu(static říká, že metoda nepotřebuje pro své fungování žádný konkrétní existující objekt, s nímž by pracovala, To přesně potřebujeme u main, neboť žádný objekt dosud nemáme). Nestatické metody jsou vázané na objekt.<br>
Instance</b> – Je objekt. To samé jako object => instance == object<br>

<b>Rekurze</b> – Metoda, která volá sama sebe. Každá rekurze by měla mít podmínku ukončení. <br>
Jednoduchá implementace metod, přirozenější chování, špatná čitelnost kódu, složité ladění, pokud chyba v kódu<br>
<b>Statická inicializace</b> - Je provedena když je odkazováno na třídu<br>
<b>Inicializace objektu</b>- Provádí se když se vytváří nová instance třídy<br>
<b>Zavolání kontruktoru</b> - První metoda, která se zavolá když objekt vzniká. Má za úkol objekt inicializovat.<br>
